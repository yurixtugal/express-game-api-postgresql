VIDEO-GAME API

This is an API implemented with NodeJS-Express and integrated with a Postgresql DB
This API has those endpoints:
  • API/V1/Developers (GET)
  • API/V1/Genres (GET)
  • API/V1/Platforms (GET)
  • API/V1/Generations (GET)
  • API/V1/GameVideos (GET, POST, PATCH, DELETE)
